# 22 — Brand Assets — Full Request List

Assets shipped with this PRD are sampled from the live site. For a proper rebuild, the following assets and documentation are required from CNZ. Treat this as a formal intake list to give to CNZ’s brand guardian before starting design work.

## 29.1 Logos — required variants

| **Variant** | **Format** | **Reason** |
| --- | --- | --- |
| Primary lockup, RGB | SVG | Scalable, sharp on all devices |
| Primary lockup, CMYK | SVG / PDF | Print / official docs |
| Primary lockup, single-colour black | SVG | Faxable / accessibility fallback |
| Primary lockup, single-colour white | SVG | Dark-mode / overlays |
| Short mark (without lockup line) | SVG | Tight spaces, mobile header |
| Icon-only (koru flourish) | SVG | Favicon, app icon, OG fallback |
| Favicon 16/32/48 | ICO/PNG | Browser tab |
| Apple touch icon 180×180 | PNG | iOS home screen |
| Android maskable 512×512 | PNG | Android |
| OG default image 1200×630 | PNG | Social share when no custom image |

## 29.2 Koru motifs — required variants

Currently sampled: orange, lime, cyan. Needed for completeness:

- **Coral** — suggested for Ngā Toi Māori section.
- **Plum** — suggested for Pacific Arts section.
- **Gold** — suggested for Results / Awards.
- **Navy** (reversed, for light backgrounds) — subtle decorative.
- **All as SVG** (line-art sources, not rasterised).
- **Multi-colour watermark cluster** — higher res (4000px wide) so it’s sharp as a decorative element.

## 29.3 Typography

- Confirm the brand typefaces. Current assumption: Source Serif 4 + Inter.
- If different: web-font files (WOFF2) for the specific weights used OR a licence allowing self-hosting.
- Licence terms document.
- Macron rendering confirmed in both typefaces at all used weights.

## 29.4 Colour specifications

- Brand guidelines with Pantone, CMYK, RGB, HEX values for every official colour.
- Accessibility-safe pairings documented.
- Tertiary palette for category-coding (Ngā Toi Māori, Pacific arts, etc.) agreed.

## 29.5 Photography / imagery guidelines

- Image treatment rules (colour treatment, composition, cropping).
- Cultural-safety guidelines — when an image of a taonga, marae, or tangata whenua requires permission; who to consult.
- Photographer credits — standard format.
- Artist/work attribution — standard format.
- Stock photography policy — confirmed as avoided; if ever needed, approved source list.

## 29.6 Editorial style guide

- Full voice document (section 4.4 is a start; CNZ may have a more authoritative version).
- Te reo Māori usage guide — authoritative term list with macrons.
- Plain language guide.
- Accessibility writing guide — alt text conventions, heading conventions, link-text conventions.

## 29.7 Iconography

- Confirm Lucide as the icon library (or supply a bespoke CNZ set).
- Custom icons for artforms (if a bespoke set exists) — SVG sources.
- Icon-to-concept map — what each icon is allowed to represent.

## 29.8 Templates and collateral

- Email template (Mailchimp) — header/footer matching new brand.
- Funded project acknowledgement logos — "Proudly supported by Creative New Zealand" lockup in the required variations.
- Document templates (if consistent with the site) — PDF covers, letterheads.
